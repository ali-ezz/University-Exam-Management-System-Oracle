# Contributing

Thank you for your interest in contributing to this Oracle Exam Management System.

## How to contribute

1. Fork the repository.
2. Create a branch for your change:
   ```bash
   git checkout -b feature/my-change
   ```
3. Make your changes and keep commits focused and descriptive.
4. Submit a pull request to the main repository.

## Contribution guidelines

- Keep changes limited to a single concern per branch.
- Use clear commit messages such as `docs: improve README` or `feat: add new report procedure`.
- Do not include sensitive data or credentials in commits.

## Reporting issues

Use the issue templates in `.github/ISSUE_TEMPLATE/` to report bugs or request enhancements.

## Code of conduct

This project follows a respectful collaboration model. Please keep discussions constructive and professional.
