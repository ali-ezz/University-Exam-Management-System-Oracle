# Tests

This repository is based on Oracle SQL and PL/SQL deployment scripts. Validation is performed by executing the `06_TESTING_ALL_FEATURES.sql` script after deployment.

## Manual validation steps

1. Deploy the schema and scripts in the correct order.
2. Run `06_TESTING_ALL_FEATURES.sql` as `MANAGER_ADMIN`.
3. Verify expected output and behavior for each feature.
